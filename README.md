# FitHub - Spor Salonu Yönetim ve Randevu Sistemi

Bu proje, Sakarya Üniversitesi Web Programlama dersi kapsamında geliştirilmiştir.  
Amaç, spor salonlarının ve antrenörlerin yönetimi ile üyelerin randevu almasını sağlayan bir web uygulaması sunmaktır.

## Özellikler
- Spor salonu tanımlama ve yönetimi
- Antrenör (eğitmen) yönetimi ve uzmanlık alanları
- Üye kayıt ve giriş işlemleri
- Randevu alma, onaylama ve yönetimi
- REST API ile veri sorgulama (LINQ ile filtreleme)
- Yapay zekâ ile egzersiz/diyet önerisi
- Rol bazlı yetkilendirme (Admin ve Üye)
- Bootstrap 5 ile modern arayüz

## Kullanılan Teknolojiler
- ASP.NET Core Razor Pages (.NET 8)
- Entity Framework Core
- SQL Server
- Bootstrap 5, HTML5, CSS3, JavaScript
- OpenAI API (Yapay Zeka entegrasyonu)
