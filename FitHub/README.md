# FitHub - Spor Salonu Yönetim ve Yapay Zeka Koçluk Sistemi

Bu proje, Sakarya Üniversitesi Web Programlama dersi için geliştirilmiş; spor salonu yönetimi, randevu takibi ve yapay zeka destekli antrenman planlaması sunan kapsamlı bir web uygulamasıdır.

## 🚀 Proje Hakkında
FitHub, spor salonlarının dijitalleşmesini sağlamak amacıyla geliştirilmiştir. Yöneticiler salonlarını, eğitmenlerini ve hizmetlerini yönetebilirken; üyeler istedikleri kriterlere göre randevu alabilir ve yapay zeka asistanından kişisel gelişimleri için destek alabilirler.

## 💡 Öne Çıkan Özellikler

* **Yapay Zeka (AI) Koç:** Kullanıcının fiziksel özelliklerine (boy, kilo, yaş) ve hedeflerine göre kişiselleştirilmiş antrenman programı oluşturur. (Groq API entegrasyonu).
* **Randevu Sistemi:** Üyeler, eğitmenlerin müsaitlik durumuna ve salon kapasitesine göre dinamik randevu alabilir. Çakışma kontrolü mevcuttur.
* **Yönetim Paneli (Admin):** Salon, eğitmen, hizmet ve randevu yönetiminin yapıldığı, istatistiklerin görüntülendiği panel.
* **Role-Based Authorization:** Admin ve Üye rolleriyle yetkilendirme altyapısı.
* **Modern Arayüz:** Bootstrap 5 ile geliştirilmiş, kullanıcı dostu ve "Dark Mode" (Koyu Tema) tasarımı.

## 🛠 Kullanılan Teknolojiler

* **Backend:** ASP.NET Core 8.0 MVC
* **Veritabanı:** SQL Server / Entity Framework Core (Code-First)
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **AI Entegrasyonu:** Groq Cloud API
* **Versiyon Kontrol:** Git & GitHub
